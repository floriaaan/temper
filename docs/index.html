Introduction

Purposes 📗#

caution

Be careful ! The documentation still is under work in progress.

The purposes of Temper is to provide a full open-source solution, to manage all your connected devices.

Disclaimer 🛑#

Temper is still under developpement stage. As now, we don't plan to release a stable version in Q3 2020. At this point, we don't recommended using it in production or as a daily driver.

For feedback and questions, feel free to ask on Discord : Floriaaan#1992

You can use this if :

  • ✅ You are curious about what comes next in Temper
  • ✅ You want to have full control of your applications
  • ✅ You are an open-source lover 💞

You can contribute if :

  • ✅ You have read and agree with our Code of Conduct
  • ✅ You want to contribute, of course

Technologies ⚛️#

We use React for our application, but as it is open-source, feel free to reproduce it in VueJS or Angular.

For our back-end, we use Lumen PHP framework, it is a light-weight version of Laravel. We plan to make different versions of Temper API to make every developper comfortable in its language. Or even for curiosity of doing the same thing in different language.
