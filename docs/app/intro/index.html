Application entry-point#

Temper app entry point file is located under /src/App.tsx : here.

logged and notLogged are the scaffolding of the entire app, it depends on 'if you're logged or not'.

const App: React.FC = () => {
  const [darkMode, setDarkMode] = useState(sessionStorage.getItem("darkMode") === 'true' || false);
  const handleDarkMode = (checked: boolean) => {
    setDarkMode(checked);
    sessionStorage.setItem("darkMode", `${checked}`);
    document.body.classList.toggle("dark", checked);
    
  };
  if (!sessionStorage.getItem("auth.logged")) {
    sessionStorage.setItem("auth.logged", "0");
  }
  useEffect(() => {
    handleDarkMode(darkMode);
    // eslint-disable-next-line
  }, []);
  
  auth_middleware();
  return <IonApp>{sessionStorage.getItem("auth.logged") === "1" ? logged : notLogged}</IonApp>;
};
export default App;

Organization#

Locations of different parts :

  • Components : /src/components
  • Pages : /src/pages
  • Routes : /src/routes/web.js
  • Middlewares : /src/middlewares

Routes and Middlewares are not common React components, but it helps in organizing project.
