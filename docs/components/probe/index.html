Properties 🔨#

The data props of Probe component is a JSON object, which is returned by the API, here's its structure :

NameUsageDescription
idprobe.idIdentification in database of the probe
nameprobe.nameName of the probe
stateprobe.stateGet the state of the probe (active or inactive)
categoryprobe.categorySort the probes by category
lastmeasureprobe.lastmeasureGives the last measure recorded by the probe
tokenprobe.tokenIdentification for the api calls
gpsprobe.gpsGet where the probe is located
